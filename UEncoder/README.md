# Universal Encoder

Tiny tool for fast recursive files encoding 


## Setting up

You can set global variables inside, or just input it in process

```py
# --------- Variables ---------
path = ''
ext = ''
force_encode = True
default_codec = 'windows-1251'
# -----------------------------
```

## Usage

```bash
python uencode.py
```

## Log file

All errors writing to `log.txt` file
